# Quant-Retail-Sales-Analysis-Dashboard
Executive-level Power BI dashboard for retail sales growth, profitability, forecasting, and quantitative business analysis using DAX measures and interactive visualizations.
Overview

This project is an executive-level Power BI dashboard focused on retail sales growth, profitability analysis, forecasting, and quantitative business insights.

The dashboard was designed to simulate real-world business intelligence reporting by combining:

KPI monitoring
trend analysis
forecasting
profitability analysis
regional performance tracking
quantitative DAX measures

The project emphasizes both executive storytelling and analytical decision-making.

Objectives

The main objectives of this project were:

Analyze retail business performance over time
Identify revenue and profitability trends
Detect growth patterns and anomalies
Evaluate regional and product-level performance
Build forecasting and risk-oriented insights
Present findings through an executive dashboard interface
